## 基于Typescript的React Native听书app开发(毕设)
### 技术栈:
* React native
* dva
* Typescript
* realm
* React Navigation
### 项目运行
* 安装依赖 
```javascript
npm install
```
* 运行后台服务器（注意需要先连接MongoDB）
```javascript
cd bear-yapi && node vendors/server/app.js
```
* 运行虚拟机
```javascript
npm android
```
